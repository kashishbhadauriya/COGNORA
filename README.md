📚 Cognora — AI Study Assistant

Cognora is an AI-powered study platform designed to help students learn smarter and faster. It allows users to upload study materials, convert files, and generate AI-powered summaries and quizzes.

🔗 Live Demo:
https://cognora-xi.vercel.app/

🚀 Features
🤖 AI-Powered Study Assistant


Users can upload study documents for processing and analysis.

✨ AI Text Summarization

Automatically generates short and clear summaries from uploaded documents.

🧠 Quiz Generator

Creates quiz questions based on uploaded study material to test knowledge.

🖼️ Image Converter(tesseract)

Allows users to convert images into readable text or other formats for easier study usage.

📑 PDF Converter(pdf parse)

Supports PDF file conversion and processing, enabling students to extract and analyze study material from PDFs.

🔐 User Authentication

Secure login and signup system with password hashing and session management.

📊 Dashboard

Personalized dashboard where users can access their uploaded materials, summaries, and quizzes.

🎨 Modern UI

Clean and responsive interface designed for a smooth learning experience.

Node.js | Express | MongoDB | Groq AI | Tesseract OCR | Llama 3.3 | EJS |vercel
